# 📚 CampusNotes

CampusNotes is a full-stack Notes Sharing Platform designed to streamline academic resource management for students and faculty.

## 🚀 Overview

This platform allows faculty to upload and organize notes based on academic structure, while students can easily access, view, and download study materials.

It follows a structured hierarchy:
Department → Year → Semester → Subject → Unit

## ✨ Features

- 🔐 Role-Based Authentication (Admin / Faculty / Student)
- 📤 Notes Upload and Management System
- 📥 Student Access to Notes
- 📊 Analytics Dashboard
- 🧠 AI Assistant (Student Support)
- 📝 Exam Mode for Practice
- 📱 Responsive UI

## 🛠️ Tech Stack

- Frontend: React.js, Tailwind CSS
- Backend: Node.js, Express.js
- Database: MongoDB

## 📁 Project Structure

- `/backend` → APIs, database, authentication
- `/frontend` → User interface and dashboards
